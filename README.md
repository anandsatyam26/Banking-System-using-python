# Banking-System-using-python
In this project we have created banking system using python for the following fuctions:-
   1. BANKING SERVICES
   2. LOAN SERVICES
   3. OTHER DEPOSITS
   4. QUIT


Under Banking services we are having the given options 

    1. NEW ACCOUNT
    2. DEPOSIT AMOUNT
    3. WITHDRAW AMOUNT
    4. BALANCE ENQUIRY
    5. ALL ACCOUNT HOLDER LIST
    6. CLOSE AN ACCOUNT
    7. MODIFY AN ACCOUNT
    8. EXIT
    
 Under Loan Services
 
    1.Home Loan
    2.Education Loan
    3.Car Loan
    4.Exit
    
  Under Other Deposits
  
   1. FIXED DEPOSIT
   2. RECURRING DEPOSIT
   3. Exit
